# Task Completion Checklist

When completing a coding task:

1. **Format code**: Run `cargo fmt`
2. **Check linting**: Run `cargo clippy` and address warnings
3. **Verify compilation**: Run `cargo check`
4. **Test**: Run `cargo test` (add tests if implementing new functionality)
5. **Build**: Run `cargo build` to ensure binary builds successfully
6. **Manual test**: Run the binary with test data to verify behavior
