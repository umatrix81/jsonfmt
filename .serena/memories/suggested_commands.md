# Suggested Commands

## Build and Run
- `cargo build` - build debug version
- `cargo build --release` - build optimized release version
- `cargo run` - run the tool (reads clipboard, prints to console)
- `cargo run -- save` - run with argument to save to file

## Testing
- `cargo test` - run tests (note: project currently has no tests)

## Code Quality
- `cargo clippy` - run linter
- `cargo fmt` - format code with rustfmt
- `cargo check` - check compilation without building

## Shell (Windows/Bash)
- `ls` - list files
- `cd <dir>` - change directory  
- `grep` - search in files
- `find` - find files
